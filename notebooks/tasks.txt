### Task 2 — Evaluate the Quantum IPM portfolio
The solver returns portfolio weights `w_qipm`.  
Now compute three basic portfolio metrics:
\[
\text{return} = w^T \mu
\]
\[
\text{variance} = w^T \Sigma w
\]
\[
\text{volatility} = \sqrt{\text{variance}}
\]

Code skeleton

# TODO: compute the portfolio return, variance, and volatility
ret_qipm = ...
var_qipm = ...
std_qipm = ...
print(f"\n⏱  Quantum IPM finished in {elapsed_qipm:.1f} s")

print(f"   Annual variance  : {var_qipm:.4f}")
print(f"   Annual volatility: {std_qipm:.2%}")

Hint markdown

<details>
<summary>Hint</summary>
Use:
- `w_qipm @ mu_vec` for expected return
- `w_qipm @ cov_mat @ w_qipm` for variance
- `np.sqrt(...)` for volatility
</details>

Solution markdown

<details>
<summary>Solution</summary>
```python
ret_qipm = float(w_qipm @ mu_vec)
var_qipm = float(w_qipm @ cov_mat @ w_qipm)
std_qipm = float(np.sqrt(var_qipm))
```
</details>
